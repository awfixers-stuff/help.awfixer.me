# The AJ Service's Dashboard

the AJ Services Dashboard is a project that will allow you to manage all of your subscribed services, as well as the products we have that you use, in one central location. It will be secure by default, mobile friendly and designed to provide you with the amount of information that you need, while preventing information overload.
